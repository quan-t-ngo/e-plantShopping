# Paradise Nursery

A React + Redux shopping cart application for an online plant shop.

## Features
- Browse plants by categories
- Add/remove plants from cart
- Adjust quantity dynamically
- Real-time cart total calculation
- Redux state management
- Responsive UI with navigation bar

## Tech Stack
- React
- Redux Toolkit
- JavaScript (ES6)
- CSS

## Project Structure
- App.jsx → Landing page
- ProductList.jsx → Plant catalog
- CartItem.jsx → Shopping cart
- CartSlice.jsx → Redux cart logic
- AboutUs.jsx → Company info
